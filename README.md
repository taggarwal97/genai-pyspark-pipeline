# E-Commerce Data Pipeline Project

An end-to-end Python data pipeline project that generates mock e-commerce data and utilizes PySpark to extract business insights.

## Project Structure
* `src/`: Houses core source code modules (`data_generator.py`, `spark_analytics.py`).
* `data/raw/`: Target location for newly generated CSV data files.
* `data/processed/`: Target location for analytical reporting deliverables.
